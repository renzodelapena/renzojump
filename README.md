# *RenzoJump*
<b>This is a platform game, you control the player to collect all of the spinning boxes.<b/>
<b>You can double jump!<b/>
**Controls**
> W - Run Forward
> S - Run Backwards
> D - Run Right
> A - Run Left
> Spacebar - Jump

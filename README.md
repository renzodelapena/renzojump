# *RenzoJump*
<b>Version 1.001<b/>
<b>Created by Renzo Dela Pena<b/>
<b>This is a platform game, you control the player to collect all of the spinning boxes.<b/> <br>
<b>You can double jump!<b/>
**Controls**
> W - Run Forward <br>
> S - Run Backwards <br>
> D - Run Right <br>
> A - Run Left <br>
> Spacebar - Jump
